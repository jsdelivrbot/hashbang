# hashbang

## Setup

Create a Ubuntu 16.04 box and run:

```
sudo apt-get update; sudo apt-get install -y wget; sudo wget -q -O - https://rawgit.com/richard-fairthorne/hashbang/master/deploy/setup | bash
```

I like a good 1 liner.
